# Next Steps

You've installed the starter config, installed a language server, and used the
keybindings. Here's where to go tomorrow and next week.

---

## Learn vim (30 minutes)

**`vimtutor`** — a built-in interactive tutorial. Run it in your terminal:

```bash
vimtutor
```

Or inside Neovim: `:Tutor`. Takes about 30 minutes and covers everything you
need to move, edit, and navigate comfortably.

---

## Read the built-in manual

Neovim has an excellent help system. Try these:

- `:help quickref` — one-page cheatsheet of every command
- `:help lsp` — how Language Server Protocol works in Neovim
- `:help vim.pack.add` — the built-in plugin manager this config uses

---

## Read the config

Open `init.lua` and read it top to bottom. It's 308 lines, all commented, and
organized into numbered sections. Understanding your config means you can
fix your own problems and add features without searching.

---

## Practice everywhere

Building vim muscle memory takes repetition. Install a vim-mode extension in
your browser:

- **Chrome / Brave:** [Vimium](https://vimium.github.io/)
- **Firefox:** [Tridactyl](https://github.com/tridactyl/tridactyl)
- **VS Code:** Built-in [Vim extension](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim)
